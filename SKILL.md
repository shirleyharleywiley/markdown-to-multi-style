# Markdown 样式转换器

一键将 Markdown 转换为带精美样式的 Word 或 HTML，8 种专业版式任你选。

**本工具专为 AI 产出内容设计**，常见的双语文档、中英混杂、表格等格式都能完美呈现。

> 查看 `examples/` 目录了解每种版式的转换效果

## 快速开始

```
/markdown-to-word-multi-style /path/to/article.md
```

AI 会自动分析文件名和内容，推荐最适合的版式，你确认后直接生成。

## 版式预览

| 版式 | 配色 | 特点 | 适合场景 |
|------|------|------|----------|
| **left-block** | 薄荷绿 | H1/H2 整块背景+斜切角 | 技术文档、公众号 |
| **bottom-deco** | 粉红 | 标题底部彩虹边线 | 技术博客、活泼风格 |
| **gradient-pill** | 柠檬黄 | H2 渐变胶囊按钮 | 公众号、轻量文档 |
| **stripe-docs** | 紫色 | 紧凑排版、层次清晰 | API文档，开发指南 |
| **cmd-markdown** | 酒红 | 衬线字体，大号引号 | 团队笔记、写作 |
| **notion** | 蓝色 | 极简无边框、干净 | 双语内容、简洁风格 |
| **medium** | 青绿 | 衬线字体，大留白 | 长文阅读、深度内容 |
| **linear** | 浅绿 | 近黑背景、深色主题 | 代码密集文档 |

**点击 `examples/` 目录中的文件，在浏览器中查看每种版式的实际效果。**

## 使用方式

### 单文件

```
/markdown-to-word-multi-style /path/to/article.md
```

对话式引导：读取文件 → 智能推荐 → 确认生成

### 批量转换

```
/markdown-to-word-multi-style /path/*.md
```

传入多个文件时，会列出处理方案，确认后再执行。

## 输出格式

- **Word (.docx)** — 仅限 Stripe Docs（紫色点缀）、Notion（极简素净）两种版式
- **HTML** — 8 种版式全部支持

Word 版式选择逻辑：想加点色彩 → Stripe Docs，想素净简洁 → Notion

**Word 表格显示效果极佳**，双语并列表头会自动合并为一个单元格。

输出到原文件同级目录，文件名：`{原名}-{版式名}.docx/html`

## 支持的 Markdown 语法

支持 Markdown 标准语法，以及 AI 常见的特殊格式：

- **双语文档**：中英并列的表头会自动合并
- **引用块内列表**：`> * 列表项`、`> - 列表项`

## 避坑指南（HTML 版式）

| 版式 | 避免用于 |
|------|----------|
| gradient-pill | 含表格的文档 |
| bottom-deco | 正式工作报告、商务文档 |
| left-block | 1、2级标题密集的文档 |
| linear | 代码占比 <50% 的文档 |

推荐时会主动检查避坑指南，用户坚持则尊重选择。
